# gh-aw-workshop

A 90-minute beginner workshop for agentic workflows, delivered as an Astro website with markdown teaching materials.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content map

- `src/pages/index.astro` — agenda, goals, daily loop
- `src/pages/facilitator-guide.md` — instructor guide
- `src/pages/learner-playbook.md` — participant guide
- `docs/workshop-instructions.md` — setup and delivery instructions
