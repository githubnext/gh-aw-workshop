## Learner Playbook

### Step 1: Frame your task

Use this starter template:

- Goal:
- Constraints:
- Existing files:
- Definition of done:

### Step 2: Ask for a plan first

Request a short plan before implementation. Confirm scope, then proceed.

### Step 3: Implement in small increments

- Generate one unit of work at a time.
- Review changes before moving on.
- Keep each commit focused.

### Step 4: Validate every change

At minimum, run:

- `npm run build`
- project tests (if available)
- security/dependency checks as required by your team

### Step 5: Reflect and improve

After each session, capture:

- One prompt that worked well
- One failure mode you observed
- One update to your personal workflow

### Beginner checklist

- [ ] I can describe the agentic loop.
- [ ] I can create a high-context prompt.
- [ ] I verify outputs before accepting them.
- [ ] I can recover when the agent makes a wrong assumption.
