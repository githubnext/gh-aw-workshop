# Agentic Workflows Workshop Instructions

This repo contains a beginner-first, 90-minute workshop you can run with any agentic engine.

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start local server:
   ```bash
   npm run dev
   ```
3. Open the displayed local URL.

## Workshop artifacts

- `/src/pages/index.astro` — main agenda and daily workflow
- `/src/pages/facilitator-guide.md` — instructor timeline and teaching prompts
- `/src/pages/learner-playbook.md` — participant instructions and checklist

## Suggested delivery model

- Run once as a 90-minute synchronous workshop.
- Reuse the daily workflow section for team standups.
- Ask each learner to submit one “before/after” prompt example.
